Week 7 - PCB Testing

Powered up the Raspberry Pi 3 for the first time with the image file.
Opened the terminal and ran this block of code git clone https://github.com/six0four/StudentSenseHat.git cd StudentSenseHat/firmware gcc -Wall -o traffic2B traffic2B.c -lwiringPi sudo ./traffic2B There was a loop in the C code that caused the LED on my PCB board to flash from Green to Red. Pretty cool! Then I ran this small block of code make sudo ./ghmain and got this

In tandem with the light sensor, I can get light readings on the terminal at 1 second intervals. Tested by covering the sensor with my hand and shining a flashflight to achieve both ends of the spectrum. Changed the system date to the current date using sudo date -s "1 DEC 2017 13:00:00" Modified code for Enterprise Wi-Fi in the paths /etc/network/interfaces and /etc/wpa_supplicant/wpa_supplicant.conf Downloaded the Humber Certificate.
